# 审稿意见回复指南

> 适用场景：你收到了期刊/会议的审稿意见（一般是"大修/Major Revision"或"小修/Minor Revision"），需要写回复信 + 提交修改稿。

---

## 1. 回复信的基本结构

```
1. 封面信（致谢编辑和审稿人，说明已认真修改）

2. 修改总览（按审稿人编号归纳主要修改——让编辑不用看全文就能了解你做了什么）

3. 逐条回复（审稿人原文 + 你的回复 + 修改位置 + 修改后的文本摘录）
   格式：Reviewer 1, Comment 1 → Response → Changes (Page X, Line Y)
```

---

## 2. 中文审稿回复模板

### 2.1 封面信

```
尊敬的编辑老师及审稿专家：

感谢您对本文提出的宝贵修改意见。我们认真研读了每一条意见，
并对论文进行了逐条修改。修改部分在文中以红色字体标注。

以下是逐条回复说明。
```

### 2.2 逐条回复格式

**同意意见的回复模板：**
```
审稿意见 1：[原文照录审稿意见]

回复：感谢审稿专家的这一建议。我们已在[第X页第Y行]进行了相应补充/
修改，具体如下：

[摘录修改后的文本]

（如需要）同时，我们在[第X页第Y行]也做了相应调整，以确保全文一致性。
```

**部分同意的回复模板：**
```
审稿意见 2：[原文照录审稿意见]

回复：感谢您指出的这一问题。我们认同您关于[同意部分]的核心关切。
关于[不完全同意的部分]，我们的考虑是：[简短说明理由——不要长篇辩解]。

作为折中方案，我们在[第X页第Y行]做了以下修改：
[摘录修改后的文本]
```

**不同意的回复模板（需要特别谨慎的语气）：**
```
审稿意见 3：[原文照录审稿意见]

回复：感谢审稿专家提出的这一视角，促使我们重新审视了该问题。
我们理解您关于[对方关切]的考虑。在修改稿中，我们补充了相关讨论
（第X页第Y行），阐述了[你的做法的学术依据]。

同时，我们在[局限性]部分（第X页）也承认了[相关局限]，以保持学术诚实。

需要说明的是，由于[客观原因——如数据不可获得/方法本身的边界/研究目的的限定]，
我们暂时无法[对方要求但做不了的事]。这一局限性已明确标注，供读者参考。
```

**格式修改类意见的回复模板（最简单）：**
```
审稿意见 X：[原文照录]

回复：感谢指出。已按建议修改（第X页第Y行）。
```

## 3. 英文审稿回复模板

### 3.1 封面信

```
Dear Editor and Reviewers,

We sincerely thank you for the constructive feedback on our manuscript.
We have carefully addressed each comment and revised the manuscript
accordingly. All changes are highlighted in blue in the revised manuscript.

Below, we provide a point-by-point response. Reviewer comments are
in italics, followed by our responses in plain text.
```

### 3.2 逐条回复格式

**同意型意见：**
```
Reviewer 1, Comment 1:
[Original comment in italics]

Response: We thank the reviewer for this suggestion. We have added
[details] in Section X (Page Y, Lines Z-W):

[Revised text excerpt]

We have also adjusted [related section] to ensure consistency (Page A, Lines B-C).
```

**部分同意型：**
```
Reviewer 1, Comment 2:
[Original comment in italics]

Response: We appreciate the reviewer's concern regarding [specific point].
We agree that [point of agreement] and have revised the text accordingly
(Page X, Lines Y-Z).

Regarding [point of partial disagreement], we would like to clarify that
[brief justification]. To address the reviewer's concern, we have
[compromise action] at Page A, Lines B-C.
```

**不同意型（英文版更委婉）：**
```
Reviewer 1, Comment 3:
[Original comment in italics]

Response: We thank the reviewer for raising this point, which prompted
us to reconsider our approach. We understand the concern about
[reviewer's concern].

In the revised manuscript, we have added a discussion of [your rationale]
in Section X (Page Y, Lines Z-W). We also acknowledge this as a limitation
in Section [Limitations] (Page A, Lines B-C).

We note that [objective reason you didn't do what they asked] prevents us
from [specific action]. We have explicitly stated this limitation so that
readers can interpret our findings accordingly.
```

## 4. 回复审稿意见的核心策略

### 4.1 "Yes, and" 策略

**永远不要让审稿人觉得你在敷衍或对抗。** 即使你不同意，也要：
1. 先感谢审稿人的意见（真诚感谢——不是客套）
2. 承认 ta 的关切是合理的（"我们理解您的关切"）
3. 说明你做了什么（如不能做对方要求的，说明你做的折中方案）
4. 如果完全不能做——承认这是研究的局限

### 4.2 逐条回复，不要遗漏

- 每条意见都要有对应的回复
- 多审稿人的意见如果重复 → 可以分别回复或交叉引用（"请参见我们对 Reviewer 2 Comment 4 的回复"）
- 每条回复必须标注具体修改位置（页码 + 行号）

### 4.3 修改痕迹

- 中文期刊：通常用红色字体标出修改部分
- 英文期刊：通常要求提交带 track changes 的版本，或标色的 clean version
- **提交前确认期刊的修改稿提交要求！**

### 4.4 语气

- 中文审稿回复：直接但尊重。比英文回复更简洁——不必过度感谢
- 英文审稿回复：始终保持感谢和建设性的基调。即使审稿人明显误解了你的论文，也要假设是自己的表达不够清晰——"We realize that our original description was unclear. We have revised Section X to better clarify..."

---

## 5. 常见错误

❌ **回复太短** —— "已修改" 三个字的回复等于没回。必须说明改了哪里、改了什么、为什么这样改
❌ **情绪化回复** —— 即使审稿人不公正，回复也要保持专业。情绪回复只会让你的论文被拒
❌ **只回复不改稿** —— 回复信写得再漂亮，修改稿没跟上 = 自掘坟墓
❌ **不标注修改位置** —— 编辑和审稿人没时间在 30 页论文里找你的修改。必须标注页码和行号
❌ **对审稿人的每条意见都说好但什么都不做** —— 少量意见可以礼貌提出不同意见（有充分理由），但不能每条都这样
