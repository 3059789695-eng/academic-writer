# 中国学生英文学术写作常见错误对照

> 适用场景：写完英文部分后逐条自查，或者中译英后对照检查。
> 这 50 条覆盖了中国学生（四六级→学术写作）最常见的语法、词汇和表达错误。

---

## 一、语法类错误

### 冠词 (a/an/the) —— 中文没有的概念

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 1 | We conducted **experiment**. | We conducted **an experiment**. | 单数可数名词前必须有限定词（a/an/the） |
| 2 | **The** machine learning has advanced rapidly. | **Machine learning** has advanced rapidly. | 抽象概念/学科名称不加 the |
| 3 | Results of **experiment** show that... | **The** results of **the** experiment show that... | 特指"这个实验"的结果 → 加 the |
| 4 | We used **GPT-4 model**. | We used **the GPT-4 model** / **a GPT-4-based model**. | 特指 GPT-4 → the；泛指 → a |
| 5 | **Most of students** improved. | **Most students** / **Most of the students** improved. | most + 复数名词（无 of），most of + the + 复数名词 |

### 时态 —— 什么时候用现在时、什么时候用过去时

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 6 | Previous studies **have find** that... | Previous studies **have found** that... | 现在完成时 = have/has + 过去分词 |
| 7 | We **collect** data from 200 participants. | We **collected** data from 200 participants. | 你做过的实验 → 过去时 |
| 8 | The results **are showed** in Table 1. | The results **are shown** in Table 1. | 被动语态 = be + 过去分词（show → shown） |
| 9 | Figure 1 **is presenting** the architecture. | Figure 1 **presents** the architecture. | 描述图表用一般现在时，不用进行时 |
| 10 | Smith (2023) **has proposed** a method. | Smith (2023) **proposed** a method. | 引用具体文献的发现 → 过去时（或 Smith's method **is**...用现在时描述方法本身） |

**时态速记表：**

| 内容 | 时态 | 例子 |
|------|------|------|
| 已知事实/普遍真理/描述你自己的图表 | 一般现在时 | "Figure 1 shows..." |
| 你做的实验/数据采集/分析 | 一般过去时 | "We collected data..." |
| 文献综述中的发现（引别人） | 现在完成时或一般过去时 | "Previous studies have shown..." / "Smith (2023) found..." |
| 文献综述中的一般性陈述 | 一般现在时 | "LLMs have been widely applied in..." |

### 主谓一致

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 11 | The data **shows** that... | The data **show** that... | data 是复数（单数是 datum，但很少用） |
| 12 | The number of participants **are** 200. | The number of participants **is** 200. | the number of + 复数名词 → 谓语用单数 |
| 13 | A total of 500 samples **was** collected. | A total of 500 samples **were** collected. | a total of + 复数名词 → 谓语用复数 |
| 14 | Neither the model nor the baseline methods **was** able to... | Neither the model nor the baseline methods **were** able to... | neither...nor 就近一致（离谓语最近的是 methods） |

### 单复数

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 15 | We reviewed many **literature**. | We reviewed many **studies** / a large body of **literature**. | literature 不可数，不能说 "many literatures" |
| 16 | We need more **evidences**. | We need more **evidence**. | evidence 不可数 |
| 17 | Many **researches** have shown... | Much **research** has shown... | research 不可数（"researches" 几乎不存在） |
| 18 | The **informations** were collected. | The **information** was collected. | information 不可数 |
| 19 | Two **software** were compared. | Two **software tools** / **pieces of software** were compared. | software 不可数 |

## 二、句式类错误

### 中式长句

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 20 | The system, which was designed based on GPT-4, and was trained on a dataset of 10,000 programming exercises collected from an online judge platform, and was evaluated by 80 students from three universities, showed significant improvement. | The system was designed based on GPT-4 and trained on 10,000 programming exercises from an online judge platform. We evaluated it with 80 students from three universities. The results show significant improvement. | 中文习惯一句话塞很多内容——英文要拆！一个句子 1-2 个信息点，超过 3 个 → 拆！ |
| 21 | Because the model is large and the training data is limited, so the performance is not ideal. | Because the model is large and the training data is limited, the performance is not ideal. | because 和 so 不能连用（中文"因为...所以"可以连用，英文不行） |
| 22 | Although we tried different parameters, but the results did not improve. | Although we tried different parameters, the results did not improve. | although 和 but 不能连用（中文"虽然...但是"可以，英文不行） |

### 主动/被动混用

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 23 | The experiment **was conducted by us**. | We **conducted the experiment**. | "by us" 很别扭——直接用主动语态 |
| 24 | **We can be seen** in Figure 1 that... | **As can be seen** in Figure 1, ... | 永远不要写 "we can be seen" |
| 25 | Every sentence starts with "We". We did this. We found that. We analyzed... | Mix active and passive: "We collected data...The data were then analyzed...Results indicate..." | 适当混合被动语态，避免每句都以 We 开头 |

## 三、词汇类错误

### 近义词误用（suggest / indicate / demonstrate / prove / reveal）

| # | ✅ 正确用法 | 📝 区别 |
|---|-----------|---------|
| 26 | The results **suggest** that... | 最安全、最常用的表达。表示"数据指向这个结论"，不强硬 |
| 27 | The findings **indicate** that... | 比 suggest 稍强，但仍不绝对 |
| 28 | Our experiments **demonstrate** that... | 证据充分时才用。比 suggest/indicate 确定得多 |
| 29 | ❌ The results **prove** that... | **学术写作中几乎不用 prove**——很少有什么能被"证明" |
| 30 | The analysis **reveals** a pattern... | 强调"揭示出之前未知的东西" |

**一句话总结：** 不确定用 **suggest**，比较确定用 **indicate**，非常确定用 **demonstrate**，永远不要用 **prove**。

### 常见用词错误

| # | ❌ 错误 | ✅ 正确 | 📝 解释 |
|---|--------|--------|---------|
| 31 | This method **is benefit** for... | This method **is beneficial** for... **/ benefits**... | benefit 是名词或动词，不是形容词 |
| 32 | We **make** an experiment. | We **conduct / perform / run** an experiment. | "做实验" 不能用 make |
| 33 | We **make** a comparison. | We **draw / conduct** a comparison. | or simply "We compare..." |
| 34 | The accuracy is **high**. | The accuracy is **high** / The error rate is **low**. | accuracy 高是对的，但 error 不能说 "high accuracy" |
| 35 | It **costs** 3 hours. | It **takes** 3 hours. | cost 用于金钱，take 用于时间 |
| 36 | The model **learns** the data. | The model **is trained on** the data. | 模型不是"学习"数据，而是"在数据上被训练" |
| 37 | We **use** a questionnaire **to** participants. | We **administer** a questionnaire **to** participants. | administer = 发放/实施（问卷、测试） |
| 38 | There **has** 200 participants. | There **are** 200 participants. | "有"在不同语境对应不同的英文，there has 永远不对 |
| 39 | **Compare with** the baseline, our method... | **Compared with** the baseline, our method... | 主语（our method）是被比较的 → 用过去分词 compared |
| 40 | **Based on** the results, we analyzed... | **On the basis of** the results, we analyzed... | "Based on" 在句首常修饰错主语。更安全：Based on these findings, **our conclusion** is... |

## 四、语用类错误

### 口语化 / 过于随意

| # | ❌ 太口语化 | ✅ 学术表达 |
|---|-----------|-----------|
| 41 | A **lot of** studies... | **Many** studies... / **A considerable number of** studies... |
| 42 | This model works **pretty well**. | This model performs **effectively** / **achieves strong performance**. |
| 43 | The results are **really** good. | The results are **highly promising** / **notably strong**. |
| 44 | **So,** we decided to... | **Therefore,** we decided to... / **Consequently,** ... |
| 45 | **Anyway,** the model still works. | **Nevertheless,** the model performs adequately. / **Despite this limitation,** ... |
| 46 | We **didn't** find any difference. | We **did not** find any significant difference. |
| 47 | This is **kind of** similar to... | This is **somewhat** similar to... / This bears **a degree of** similarity to... |
| 48 | The system **can't** handle large inputs. | The system **cannot** handle large inputs. / The system is **unable to** process large inputs. |

> ⚠️ **学术写作不用缩写：** don't → do not, can't → cannot, isn't → is not, won't → will not。这不是语法错误，但会被视为不正式。

### 语气过强 / 过度自信

| # | ❌ 太绝对 | ✅ 恰当缓和 |
|---|---------|-----------|
| 49 | This method **solves** the problem completely. | This method **addresses** the problem / **provides a promising solution for** the problem. |
| 50 | There is **no doubt** that... | The evidence **strongly suggests** that... |

> ⚠️ **学术写作的黄金法则：** 承认不确定性不是弱点，是严谨的表现。用 suggest/indicate/may/might/possible/likely 而不是 prove/show conclusively/certainly/no doubt。

## 五、快速自查清单

写完英文段落或翻译完后，按这 10 条快速扫一遍：

- [ ] 每个单数可数名词前面有 a/an/the 了吗？
- [ ] 自己做的实验用的是过去时吗？
- [ ] 图表描述用的是现在时吗（"Figure 1 shows..."）？
- [ ] 每个句子主语和谓语单复数一致吗？
- [ ] 长句（超过 25 个单词）拆成短句了吗？
- [ ] because 没和 so 连用、although 没和 but 连用了吧？
- [ ] 没有 "prove"（全改成 suggest/indicate/demonstrate）吧？
- [ ] 没有缩写（don't/can't/isn't → do not/cannot/is not）吧？
- [ ] "many literatures/researches/evidences" 全改成单数了吧？
- [ ] 每段话至少用了一个缓和语（suggest/may/possible）吗？
